# VoltWise Agent

Local agent responsible for connecting EV chargers to the VoltWise cloud.

Runs on-site (hotel, parking, etc).

## 🧠 Responsibilities

- Communicate with chargers (OCPP)
- Sync data with cloud
- Handle offline scenarios
- Local caching

## 🛠️ Tech Stack

- Go (planned)
- or Python (TBD)

## 🌐 Role

Acts as a bridge between local infrastructure and the cloud.

## 🚀 Goal

Ensure reliable and resilient communication with chargers.
